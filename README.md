# Rainbow Entertainment — K-Culture Platform

Official landing page for Rainbow Entertainment (㈜빛나는별 · NGOI SAO SANG VINA).

> Where music gathers, brands grow. Rainbow connects K-culture experiences, commerce and media.

## 사이트

- **Production:** https://rainbowent.live
- **Hosting:** GitHub Pages (`index.html`)
- **Domain:** GoDaddy → GitHub Pages (apex `rainbowent.live` + `www`)

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 다국어(KO/EN/VI) 단일 페이지 랜딩 — 폰트·스타일·스크립트 인라인 포함 |
| `CNAME` | GitHub Pages 커스텀 도메인 설정 (`rainbowent.live`) |

## 로컬에서 보기

별도 빌드 없이 `index.html`을 브라우저로 열면 됩니다.

## 배포

`main` 브랜치에 푸시하면 GitHub Pages가 자동으로 반영합니다.
